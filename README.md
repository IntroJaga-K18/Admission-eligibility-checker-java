# College Admission eligibility checker
It is a simple beginner level java project. It is used to check the eligibility of a student to get admission in a college.

# what this program does??
1) Take the user inputs like Age,entrance exam status(pass/fail) , admission type, family income status(low/medium/high), and gender as well.
2) Based on family status and gender , it will declare whether the student is eligible for fee concession or not.
3) At last it will end the program by giving a declaration about the eligibility in that college.
# what concepts i have used??
I have used simple and beginner level concepts in java programming language i.e,
taking user input
using if-else, and nested conditions
string comparision as well
